# MusicLite

A lightweight music player app built with React Native and Expo.

## Features

- Clean, minimalist UI
- Create and manage playlists
- Download songs for offline listening
- Search for music
- Track listening history and favorites
- Dark and light theme support

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install` or `bun install`
3. Start the development server with `npm start` or `bun start`

## GitHub Integration

This project includes a script to easily upload changes to GitHub:

```bash
# Make the script executable
chmod +x github-upload.sh

# Run the script to upload changes
./github-upload.sh
```

## Tech Stack

- React Native
- Expo
- Zustand for state management
- Expo Router for navigation
- AsyncStorage for persistence
